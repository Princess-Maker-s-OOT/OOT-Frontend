# 🌈 OOT (Outfit Of Today) – 의류 거래 & 디지털 옷장 플랫폼 | Frontend

Next.js 기반으로 구현된 **의류 거래 + 디지털 옷장 관리** 웹 서비스입니다.  
사용자는 옷을 등록하고, 판매하거나, 기부처를 찾고, 실시간 채팅으로 거래까지 이어갈 수 있습니다.

---

# ✨ 주요 기능 (Features)

## 🛍️ 거래(Trading)
- 판매글 등록/수정/삭제
- 이미지 업로드(S3 Presigned URL)
- 카테고리·가격 설정
- **실시간 채팅 (WebSocket + STOMP)**
- **토스 페이먼츠 결제 연동**

## 👕 디지털 옷장(Closet)
- 옷 등록(사이즈/색상/카테고리)
- 공개/비공개 설정 (소셜 옷장)
- 착용 빈도 기반 **통계 대시보드**
- 카테고리별 분포 시각화

## 🎁 기부(Donation)
- **카카오맵 기반 기부센터 검색**
- 기부 내역 확인

## 👤 사용자(User)
- 회원가입·로그인(JWT + OAuth2 Google)
- 프로필 수정
- 로그인된 디바이스 관리
- 계정 보안 설정

---

# 🛠️ 기술 스택 (Tech Stack)

### Frontend
- **Next.js 15.3.3 (App Router)**
- **TypeScript**
- **Tailwind CSS**
- **shadcn/ui**
- **Recharts**

### API & Infra
- REST API + WebSocket(STOMP)
- JWT 인증
- Kakao Maps API
- Toss Payments
- AWS S3 이미지 업로드
